
# 🧠 SecondMind

### عقلك الثاني الذي لا ينسى - مدير مهام بالذكاء الاصطناعي

تطبيق موبايل احترافي مبني بـ Flutter يساعدك على تحويل الفوضى الرقمية إلى نظام منظم من المهام والتذكيرات، باستخدام أحدث تقنيات الذكاء الاصطناعي.

---

## 📖 عن المشروع

**SecondMind** هو تطبيق إدارة مهام ذكي يستخدم تقنيات الذكاء الاصطناعي لاستخراج تفاصيل المهام من النصوص والصور والإعلانات، وتحويلها إلى تذكيرات منظمة.

يقدم التطبيق تجربة سلسة وحديثة تساعد المستخدمين على:
- توفير الوقت في إدخال المهام يدوياً
- عدم نسيان الفعاليات والمواعيد المهمة
- تنظيم الحياة اليومية بشكل احترافي

يعكس هذا المشروع خبرة قوية في:
- 📱 تطوير تطبيقات الموبايل بـ Flutter
- 🎨 تصميم UI/UX احترافي (Glassmorphism + Neomorphism)
- 🏗️ Clean Architecture وكود نظيف قابل للتوسع
- 🤖 تكامل نماذج الذكاء الاصطناعي (Gemini API)
- 💾 تخزين محلي عالي الأداء (Hive)

---

## ✨ المميزات الرئيسية

### 🤖 الذكاء الاصطناعي
- تحليل النصوص واستخراج التفاصيل تلقائياً
- قراءة النصوص من الصور (OCR)
- مسح رموز QR لاستخراج المعلومات
- توصيات ذكية لتحسين الإنتاجية

### 📋 إدارة المهام المتقدمة
- إضافة، تعديل، حذف المهام
- تصنيف حسب الأولوية (عاجل، عالي، متوسط، منخفض)
- تصفية حسب الحالة (جديد، قيد التنفيذ، مكتمل)
- مهام فرعية مع وقت تقديري
- مرفقات وملفات

### 📊 الإحصائيات والتحليلات
- رسوم بيانية للإنتاجية الأسبوعية
- تحليل المهام حسب الفئة والوقت
- تتبع الإنجازات والأيام المتتالية
- تقارير أداء ذكية

### 🎯 وضع التركيز العميق
- مؤقت بومودورو مع عداد تنازلي
- حلقة تقدم دائرية متحركة
- أصوات محيطة (مطر، غابة، أمواج، تأمل، Lo-fi)
- نصائح تحفيزية ذكية
- إحصائيات جلسات التركيز

### 🔔 تذكيرات ذكية
- إشعارات للمهام القادمة والمستحقة
- تذكيرات سياقية بناءً على الجدول
- ملخص يومي للإنجازات
- تحديد وقت التذكير

### 🌙 تجربة مستخدم متطورة
- تصميم Glassmorphism + Neomorphism
- الوضع المظلم للاستخدام الليلي
- دعم كامل للغة العربية (RTL)
- تأثيرات حركية سلسة

### 🔒 الخصوصية والأمان
- تخزين بياناتك محلياً على جهازك فقط
- تصدير البيانات للنسخ الاحتياطي
- لا نشارك بياناتك مع أطراف ثالثة

### 📤 ميزات إضافية
- مشاركة المهام مع الآخرين
- ربط مع تقويم Google (قريباً)
- البحث المتقدم في المهام
- تصدير المهام كملف JSON

## 🚀 طريقة التثبيت والتشغيل

### 📋 المتطلبات الأساسية

قبل تشغيل المشروع تأكد من توفر:
- ✅ Flutter SDK مثبت
- ✅ Android Studio أو VS Code
- ✅ Git مثبت
- ✅ محاكي Android/iOS أو جهاز حقيقي

### ⚙️ خطوات التثبيت

**1. استنساخ المستودع**
```bash
git clone https://github.com/your-username/secondmind_app.git
```

**2. الدخول إلى مجلد المشروع**
```bash
cd secondmind_app
```

**3. تثبيت الحزم والمكتبات**
```bash
flutter pub get
```

**4. تشغيل التطبيق**
```bash
flutter run
```

### 🔧 بناء التطبيق للإصدار

```bash
# بناء APK لنظام Android
flutter build apk --release

# بناء AAB لنظام Android
flutter build appbundle --release

# بناء لتطبيق الويب
flutter build web --release
```
