## 📖 عن المشروع

**SecondMind** هو تطبيق موبايل احترافي مبني بـ **Flutter** يستخدم قوة **الذكاء الاصطناعي (Gemini API)** لاستخراج تفاصيل المهام من النصوص والصور والإعلانات، وتحويلها تلقائياً إلى تذكيرات وخطط منظمة.

> 💡 **فكرة المشروع:** في عالم مليء بالإشعارات والإعلانات والمنشورات، يساعدك SecondMind على التقاط كل شيء مهم وتنظيمه دون عناء يدوي.

يعكس هذا المشروع خبرة قوية في:
- 📱 تطوير تطبيقات الموبايل بـ Flutter
- 🎨 تصميم UI/UX احترافي (Glassmorphism + Neomorphism)
- 🏗️ Clean Architecture وكود نظيف قابل للتوسع
- 🤖 تكامل نماذج الذكاء الاصطناعي (Gemini API)
- 💾 تخزين محلي عالي الأداء (Hive)

## 🚀 طريقة التثبيت والتشغيل

### 📋 المتطلبات الأساسية

قبل تشغيل المشروع تأكد من توفر:

- ✅ **Flutter SDK** مثبت ([دليل التثبيت](https://flutter.dev/docs/get-started/install))
- ✅ **Android Studio** أو **VS Code** مع إضافة Flutter
- ✅ **Git** مثبت
- ✅ محاكي Android/iOS أو جهاز حقيقي

### ⚙️ خطوات التثبيت

```bash
# 1️⃣ استنساخ المستودع
git clone https://github.com/Alhayek7/secondmind_app.git

# 2️⃣ الدخول إلى مجلد المشروع
cd secondmind_app

# 3️⃣ تثبيت الحزم والمكتبات
flutter pub get

# 4️⃣ تشغيل التطبيق
flutter run
```

### 🔧 بناء التطبيق للإصدار

```bash
# بناء APK لنظام Android
flutter build apk --release

# بناء AAB لرفع على Google Play
flutter build appbundle --release

# بناء لتطبيق الويب
flutter build web --release

# بناء لنظام iOS
flutter build ios --release
```
